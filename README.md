# 台股 財報健檢 + 月營收追蹤

個人投資工具。全繁體中文、純前端、資料存在本機瀏覽器。

## 檔案
- `index.html` — 主程式(雙擊即可開,手機也能用瀏覽器開)。
- `fetch_data.py` — 自動抓官方資料的腳本(階段二)。
- `data.js` / `data.json` — 抓回來的資料(由腳本產生;前端讀 `data.js`)。

## 三個功能
1. **財報健檢** — 手動輸入財報數字,自動算毛利率/營益率/淨利率/負債比/流動比/速動比/PE/PB,亮紅綠燈並白話解釋。
2. **月營收追蹤** — 每檔每月營收 + YoY/MoM,判讀 加速🟢/退燒🟡/衰退🔴。
3. **決策日誌** — 每檔每月寫一行判斷,本機持久化。

## 自動更新資料(階段二 A:本機半自動)
一天只跑一次(官方 API 有頻率限制):

```bash
cd ~/股票健檢
python3 fetch_data.py
```

會更新 `data.js` / `data.json`,重新整理 `index.html` 即可看到最新「自動資料總覽」,並可一鍵「帶入自動抓的最新月」「帶入自動股價」。

## 資料來源(官方一手)
- 上市:TWSE OpenAPI（月營收、每日收盤)
- 上櫃:櫃買中心 TPEx OpenAPI（聯亞 3081、上詮 3363 為上櫃股)

## 正確性把關(四道防線)
1. 官方一手來源(上市 TWSE、上櫃 TPEx)。
2. YoY 交叉驗證:API 值 vs 自算值,差 > 0.5 個百分點標記。
3. 合理性範圍檢查:營收非負、YoY −100%~999%、股價 > 0。
4. 來源 + 抓取時間戳記;抓不到留空,不用舊值亂填。標「⚠️ 需確認」者請人工核對。

## 注意
- 財報那 16 格屬季報(MOPS),目前仍手動填,最可靠。自動只負責「月營收 + 股價」。
- 換裝置或清瀏覽器快取前,用 app 內「匯出備份」保存你的日誌與輸入。
